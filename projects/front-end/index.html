what is front end devolopment ?

front end devolopment is to create websites and applications using web languages such as HTML, CSS, and JavaScript that allow users to access and interact with the site or app.

Front end devoloper skills

Front-end developers will typically possess a variety of technical skills to help develop and designwebsites. While a large part of your responsibilities as a developer revolves around building a user-friendly interface non-technical skills can contribute to your success in this field as well

Technical skills

Here some technical skills you need as Front-end devolper

  1. HTML, CSS, and JavaScript: These three front-end development languages are he foundation of web design and are the basic languages you should know

  2. SQL and NoSQL: Front-end developers need to be able to interact with permanently stored data such as user profile information

  3. Testing and debugging: Front-end developers perform unit testing and functional testing

Workplace skills

Focused on the visual aspect of a website, front-end developers possess workplace skills that benefit the user experience which include:

  1. Creativity: Designing an aesthetically pleasing website layout means you will need to be creative and innovative.

  2. Problem-solving: In web development, you are bound to face a challenging problem, whether hat be identifying and correcting coding errors or fixing bugs
